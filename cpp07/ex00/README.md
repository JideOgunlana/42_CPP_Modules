<h2><u>Summary</u></h2>

> *ex00 creates a program named \"Whatever\" that uses template functions max, min and swap. These functions are scoped globally and only takes similar data types*

<br>

```cpp
    ::max(10, 20); // 20
    ::min('a', 'b'); // a

    int x = 0;
    int y = 5;
    ::swap(x, y); // x = 5, y = 0
```
